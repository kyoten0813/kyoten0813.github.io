初心者向けGitHub入門

GitHubの基本と他サービスとの違いを分かりやすく解説！

1. GitHubの基本的な使い方

  • リポジトリの作成方法
    1. GitHubにログインし、右上の「＋」ボタンから「New repository」を選択します。
    2. リポジトリ名を入力し、「Create repository」をクリックします。
  • コミット
    ファイルを編集したら、git add ファイル名 で変更を追加し、git commit -m "メッセージ" で保存します。
  • プッシュ
    ローカルの変更をGitHubに反映するには、git push を実行します。
  • GitHub Pagesの公開方法
    1. リポジトリの「Settings」→「Pages」を開きます。
    2. 「Branch」で公開したいブランチ（例: main）を選択し、「Save」をクリックします。
    3. 数分後、指定されたURLでページが公開されます。

2. GitHubと他サービスの比較

サービス名 特徴 主な用途
GitHub 世界最大のOSS開発プラットフォーム。
無料で公開リポジトリが作成可能。
GitHub Pagesで静的サイトも公開できる。		 個人・チームの開発、OSS公開、ポートフォリオ作成
GitLab 自社サーバーにインストール可能。
CI/CD機能が強力。		 企業・組織内の開発、プライベートプロジェクト
Bitbucket Atlassian製。
プライベートリポジトリが無料で利用可能。		 小規模チーム、Jira連携

3. まとめ

GitHubは初心者にも使いやすく、学習やポートフォリオ作成にも最適です。
まずはリポジトリを作成し、ファイルをアップロードしてみましょう。
分からないことがあれば、公式ドキュメントやコミュニティも活用してみてください！
